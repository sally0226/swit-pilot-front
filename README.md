## Installation

### Environment

> OS : Windows10 x64
>
> Node.js : v16.13.0(LTS)
>
> npm : v8.1.0
>
> yarn : v1.22.17

### Dependencies

```shell
yarn install # 또는 yarn
```

## Run

```shell
yarn start
```
